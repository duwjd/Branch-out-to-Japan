# 00 · 포지셔닝

> 이 문서는 팀 전원과 모든 에이전트가 공유하는 **전략의 기준점**이다.
> 카피·기획·디자인·개발 의사결정이 흔들릴 때 이 문서로 돌아온다.

## 한 줄 정의

한국 뷰티 브랜드의 상세페이지·SNS·광고 문구를 **일본 고객 관점으로 진단**하고,
일본향 **카피·콘텐츠·운영**을 한곳에서 다루는
**일본향 메시지 설계가 내장된 Japan Growth Studio — 뷰티 중소기업용 내부 운영 툴 웹 서비스**.

> 진단(리포트)에서 그치지 않고, 일본향 콘텐츠 **제작**과 **운영**까지 잇는 3축 서비스로 빌드한다.
> 상품 구조: [[decisions/2026-07-04-상품-구조-구체화]]

## 최종 포지션

**일본 진출 전 콘텐츠 진단 & 일본향 콘텐츠 제작·운영 서비스** (3축: ① 진단 리포트 · ② 마케팅 스튜디오 · ③ 운영)

- 직관 메시지 A: "번역만으로는 부족한 일본 진출 콘텐츠, 지금 일본에서 통할지 진단해드립니다."
- 직관 메시지 B: "일본 진출 전, 지금 콘텐츠가 일본 고객에게 통할지 먼저 점검하세요."

> ※ 이전 "광고 소재 스타터 키트 / 7일 컨시어지" 표현은 3축 직접 개발 결정(2026-07-04, [[decisions/2026-07-04-상품-구조-구체화]])으로 대체됨.

## 핵심 차별점 (한 문장)

> Carat은 광고 영상을, 피카클립은 숏폼을, AlphaCut은 채널 분석을, Buffer는 게시를 해준다.
> **우리는 일본 고객에게 팔릴 "말"과 콘텐츠 방향을 정해준다.**

번역이 아니라 **일본 고객 관점의 메시지 재설계**가 우리의 정체성이다.

## 우리가 가져갈 것

- 일본 고객 관점의 USP·구매 이유 재설계
- 대행사에 큰 비용을 쓰기 **전**의 사전 진단 상품
- 일본향 **메시지 설계·페르소나·USP 재정의가 내장된 도구를 직접 만든다** (범용 자동생성툴과 다른 지점)

## 우리가 절대 피해야 할 포지션

| 피해야 할 포지션 | 이유(정면 경쟁 상대) |
|---|---|
| AI 광고 영상 생성툴 | Carat, Arcads, AdCreative.ai |
| 쇼츠 자동 변환 서비스 | 피카클립, AlphaCut, OpusClip |
| 유튜브 채널 분석기 | vidIQ, TubeBuddy, AlphaCut |
| SNS 예약 발행툴 | Buffer, Metricool, Hootsuite |
| 일본어 번역 서비스 | DeepL, ChatGPT, 프리랜서(가격 경쟁) |
| 일본 진출 종합 대행 | AnyMind, transcosmos, @cosme, 현지 대행사 |

> 위 6개 포지션은 **금지선**이다. 모든 에이전트는 산출물이 이 방향으로 흐르면 즉시 교정한다.
>
> ※ 우리도 도구를 **만든다**. 다만 범용 자동생성/번역이 아니라, **일본 고객 관점의 메시지 재설계가 내장된 점**이 위 경쟁사와의 경계선이다.

## 발견한 빈자리

한국 소형 뷰티 브랜드가 처음 막히는 지점:
> "일본 고객에게 어떤 말로 팔아야 할지 모르겠다. 한국식 상세페이지를 그대로 번역해도 되나? 어떤 콘텐츠를 올려야 하지? 대행사에 맡기기 전에 통할 가능성이 있는지 먼저 알고 싶다."

→ 단순 제작툴·번역툴이 해결하지 못하는 이 지점이 우리의 자리다.
