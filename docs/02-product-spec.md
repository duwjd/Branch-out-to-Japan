## 커머셜 모델 (검증 가설)

> 아래 3단계 상품/가격은 **초기 검증 가설**이다. 빌드하는 3축 서비스에 어떤 형태(무료 진입 → 유료 심화)로 얹을지는 **개발·UT(8월) 이후 확정**한다.

### 1단계 · 무료 리드 상품 — "일본 진출 콘텐츠 5분 진단"

입력받을 정보:

| 입력 항목 | 목적 |
|---|---|
| 브랜드명 / 제품명 | 카테고리 파악 |
| 제품 상세페이지 URL | 카피·구조 진단 |
| 인스타그램 계정 | 콘텐츠 톤 진단 |
| 일본 진출 목표 | Qoo10 / 인스타 / 유튜브 / 오프라인 구분 |
| 현재 고민 | 실제 고객 언어 수집 |

제공 결과:

| 무료 결과 | 목적 |
|---|---|
| 일본향 콘텐츠 위험도 5개 | 문제 인식 |
| 고쳐야 할 문구 예시 3개 | 전문성 인식 |
| 추천 콘텐츠 방향 3개 | 상담 전환 |
| 유료 진단 제안 | 매출 전환 |

### 2단계 · 유료 스타터 — "생존자들 Starter Kit"
가격 테스트: 49 / 79 / 120만 원.

산출물: 일본향 브랜드 메시지 진단, 상세페이지 카피 개선, SNS 콘텐츠 기획 10개, 광고 카피 5개(문제형·결과형·신뢰형·비교형), 숏폼 콘티 3개, 샘플 광고 소재 1~3개, 실행 우선순위(Qoo10/인스타/유튜브/랜딩 중 어디부터).

### 3단계 · 월간 운영 — "Japan Content Sprint"
가격 테스트: 월 150 / 200 / 300만 원.

산출물: 월간 콘텐츠 캘린더(일본향 주제 20개), 광고 소재 월 4~8개, 숏폼 콘티 월 4개, 일본어 카피, 성과 리뷰, 다음 달 개선안.

## MVP 성공 기준 (검증 지표)

| 지표 | 판단 기준 |
|---|---|
| 광고 클릭률 | 문제 문구에 반응하는가 |
| 랜딩 신청률 | 제안이 이해되는가 |
| 무료 진단 신청 수 | 관심이 충분한가 |
| 상담 전환 수 | 문제의 긴급도가 있는가 |
| 유료 전환 | 실제 사업화 가능성 |

유료 상품 최소 성공 기준(예): 2주 내 무료 진단 신청 10건+, 상담 전환 3건+, 유료 결제(의사) 1건+, 반복되는 고객 문제 문장 3개+ 확보.

빌드 서비스는 위 지표에 더해 **UT 기반 검증(8/1~8/3)** — 실제 사용자가 3축 한 사이클(입력 → 리포트 → 콘텐츠 생성)을 완주하는지, 산출물이 "일본에서 통할 것 같다"는 신뢰를 주는지 — 로 확인한다.
