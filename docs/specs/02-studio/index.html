일본향 썸네일 스튜디오 — 화면 흐름

한국 썸네일 1장 업로드 → 템플릿 8종 중 1개 선택 → 생성. 생성 로딩은 별도 화면이 아니라 스튜디오 홈 피드의 타일 상태입니다. 제출하면 홈으로 돌아오고, 그리드 맨 앞의 생성중 타일이 완성 썸네일로 바뀝니다. 홈에서 시작해 전체 플로우를 클릭으로 체험하세요.

CSS 목업 고지. 화면 속 썸네일(갤러리·템플릿 프리뷰·결과 이미지)은 전부 CSS로 그린 목업이며 실제 생성물·실측 이미지가 아닙니다. 예시 브랜드는 가상 브랜드 HARUON이고, 카피·해설 예시는 정본 스펙(../02-thumbnail-converter-spec.md §3 골든 픽스처)에서 치환했습니다. gpt-image-2.0 실 API 검증은 미실시(스튜디오 주간 7/18~24 예정)입니다.

전체 플로우 — 홈이 허브다

①리포트의 "입력→처리→결과" 직렬 구조와 달리, 스튜디오는 홈(갤러리)을 허브로 돕니다. 생성 제출은 홈으로 돌아오고, 완료 확인과 결과 진입이 모두 홈에서 일어납니다.

1 · 홈

스튜디오 홈

갤러리 · 빈 상태 온보딩. "새 썸네일 만들기"로 진입.

 → 2 · 생성

썸네일 생성

업로드 1장 + 템플릿 8종 + 플랫폼 추천 + 프루프 게이트.

 → ↩ 다시 홈

생성중 타일 → 완료

제출 즉시 홈 회귀. 타일이 5단계 진행 후 완성본으로 전환 + 토스트.

 → 3 · 결과

생성 결과 상세

큰 이미지 + 재설계 해설 3단 + PNG 다운로드(실동작).

생성중에도 다른 화면으로 이동할 수 있습니다(처리는 계속됨). 홈의 검토용 상태 스위처로 빈/기본/생성중/실패 4상태를 바로 볼 수 있습니다.

템플릿 8종 ↔ 플랫폼 매핑
