원본 (Before)
한국어 카피·특가 뱃지·형광 테두리가 있는 프로모 썸네일

왜 이 문법인가

라쿠텐 공식샵 기준으로, 일본 구매자가 첫눈에 훑는 신뢰 마커(公式ショップ 칩·기능 칩·하단 카피 밴드)를 갖춘 공식샵 문법을 적용했습니다. 표본 120장에서 라쿠텐 공식샵 썸네일의 전형으로 관찰된 구조입니다.

카피는 이렇게 재설계됐다

자동 번역이 아니라 의도를 추출해 일본 고민 어휘·관례어로 다시 썼습니다. 薬機法 1차 스크리닝 통과분만 이미지에 들어갑니다.

「白浮きしない、透明感トーンアップUV」

KR 원문 쿨톤 치트키. 밈 어휘라 일본 고객에게 의미가 전달되지 않음

의도 추출 "하얗게 들뜨지 않는 톤업"

재설계 근거 白浮き = 코퍼스에서 확인된 일본 고민 어휘 · トーンアップUV = 선케어 관례어

「SPF50+ / PA++++ / 顔・からだ用」

KR 원문 얼굴부터 바디까지 OK

재설계 근거 일본 선케어의 관례 표기 顔・からだ用으로 교체. SPF·PA는 패키지 인쇄 사실만 칩으로 노출

실적을 입력하지 않아 랭킹·수상 배지 없이 생성됐습니다. 배지 없음이 기본값이고, 근거를 입력하면 배지가 들어갑니다.
