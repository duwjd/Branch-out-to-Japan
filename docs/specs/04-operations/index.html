③ 운영 · 인터랙티브 프로토타입

자산 라이브러리·브랜드 관리·기업 매칭 — 화면 흐름

③ 운영은 생성의 입력을 정리하고 그 결과를 보관하는 축입니다. 생성의 입력이 되는 브랜드 킷을 한곳에서 정리하는 브랜드 관리와, ①·②에서 만든 자산이 모이는 자산 라이브러리, 카드를 클릭하면 탭을 떠나지 않고 열리는 자산 상세, 그 자산을 들고 일본 현지 기업과 연결되는 기업 매칭까지 4개 화면으로 구성합니다(07.21 하위 메뉴 IA 개편). 사이드바 "운영"의 하위 메뉴 [브랜드 관리 / 자산 라이브러리 / 기업 매칭]로 오갑니다(2026-07-22 순서 개편 — 설정 → 보관 → 확장). "운영"을 클릭하면 첫 항목인 브랜드 관리로 진입하며, 운영 홈이라는 화면은 없습니다. 라이브러리 안에서 리포트/썸네일은 타입 탭으로 나뉩니다. 시즌 캘린더는 별도 화면 없이 자산 라이브러리 상단 스트립으로 내장했습니다.

CSS 목업 고지. 화면 속 썸네일·리포트 표지·파트너 제안 카드는 전부 CSS로 그린 목업이며 실제 산출물·실측 데이터가 아닙니다. 예시 브랜드는 가상 브랜드 HARUON입니다. 매칭 제안에는 적합도 점수·등급을 그리지 않았습니다(근거 없는 판정 수치 금지, 증거 원칙).

전체 플로우 — 모이는 화면 하나, 상세 화면 하나, 연결 화면 하나

②의 "홈=갤러리 허브" 문법을 계승합니다. 운영은 3화면 메뉴 그룹이라 중간 인덱스 화면이 없습니다. 개편 포인트: 그리드 카드를 클릭해도 더는 ①·② 탭으로 이탈하지 않고, 운영 안의 자산 상세가 열립니다 — 다른 축으로 가는 동선은 상세 화면의 버튼 2개(② 다른 템플릿·① 리포트 전체 보기)뿐입니다. 브랜드 관리·기업 매칭은 사이드바 하위 메뉴에서 병렬로 진입하는 독립 화면입니다 — 운영 홈 하단 엔트리 카드는 폐지했습니다(LIB-07 개편).

↩ 유입

자산은 ①·②에서 흘러든다

리포트·썸네일의 재조회 전용. 여기서 생성하지 않음(08-data-flow §5.2).

→ 1 · 운영 홈

자산 라이브러리

시즌 스트립 + 준비 제안 + 타입 탭 [진단 리포트|썸네일] 그리드 + 성과 판별 자리 예약.

 → 2 · 자산 상세

탭 이탈 없이 열람

썸네일: 검수 배지·재설계 해설 축약·다운로드·"다른 템플릿"(② 이동). 리포트: 표지 요약·"전체 보기"(① 이동).

사이드바 하위 메뉴에서 병렬 진입 — 운영 홈을 거치지 않아도 됩니다

3 · 브랜드 관리

브랜드 킷 = 생성 입력 허브

프로필·제품·채널 + 일본향 용어집·금지 표현·톤 가이드. 킷 수정은 발행 리포트에 소급되지 않음(스냅샷 원칙).

 4 · 기업 매칭

신청 → 검토 → 제안

컨시어지형: 신청 폼 + 상태 스테퍼. DB 열람·조건 검색 없음, 운영팀 수동 검토. 상태는 하위 메뉴 배지로 요약.

운영 홈의 검토용 상태 스위처로 빈/기본/생성중 3상태를, 브랜드 관리에서 기본/미작성 2상태를, 매칭 화면에서 폼/검토 중/제안 도착/리포트 없음 4상태를, 자산 상세 화면에서 a1~a4·r1·r2 자산 전환 데모를 바로 볼 수 있습니다. 매칭 상태는 하위 메뉴 "기업 매칭" 배지에도 반영됩니다. ② 생성 화면에서 제출하면 운영 홈 그리드 맨 앞에도 생성중 타일이 나타납니다(축 횡단).

이 기획이 다루는 것 · 다루지 않는 것

문서상 "운영"은 앱 메뉴(A)와 구독 서비스(B) 두 갈래로 쓰여 왔습니다. 이 기획은 A만 다룹니다. B의 정본은 구독 플랜 문서입니다.

영역 이 기획 비고
브랜드 자산 라이브러리실화면 ○운영 홈이 겸함. `Report`·`GeneratedAsset` 재조회 전용. 운영 축에서 유일하게 데이터 계약이 정의된 기능
시즌 캘린더홈 내장 ○별도 화면 없음. 조회 전용 스트립(예약 발행 스케줄러 아님)
브랜드 관리 (브랜드 킷)실화면 ○`BrandProfile` 편집 정본 + 일본향 용어집·톤(07.21 신설). 마이페이지는 조회 요약만. 생성 자동 참조 계약은 제안 상태(BRAND-09)
일본 기업 매칭라이트 △신청+상태 추적의 컨시어지형만. 매칭 DB·조건 검색은 넣지 않음(확정 범위)
성과 판별자리 예약실현가능성·우선순위 미정(2026-07-04 결정). 지표 UI로 그리지 않음
리뷰 축적 운영 키트 · 첫 4주 성과 리포트범위 외 ✕B계열(구독 서비스 실체). 정본 = ../03-구독플랜-재구성-제안.md (Growth 월 20만 편입 확정)
금지 포지션 경계. 시즌 스트립은 "언제 올릴지"를 예약하는 도구가 아니라 "무엇을·어떤 말로 준비할지"를 제안하는 조회 전용 영역입니다. SNS 예약 발행툴(Buffer류)·종합 운영 대행은 금지 포지션이며, 매칭 카피에도 "대행" 대신 "연결"만 씁니다.

이 프로토타입이 다루지 않은 것

확정 전 항목은 화면으로 그리지 않았습니다(결정된 것처럼 읽히지 않도록).
