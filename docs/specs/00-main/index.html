3축을 담는 현관 — 화면 흐름

로그인·온보딩 후 진입하는 앱 홈이자 ① 진단 리포트 · ② 마케팅 스튜디오 · ③ 운영의 허브입니다. 설계 원칙은 지표 대시보드가 아니라 "다음 단계 제안"입니다. 3축 성숙도가 서로 달라, 홈이 지금 할 일 하나를 골라 보여줍니다. 셸은 좌측 사이드바(브랜드 스위처·3축 내비·KPI 위젯·품의 PDF·계정) + 본문(다음 단계 밴드 → 리포트 요약·브랜드 정보 위젯 → 다가오는 이벤트 → 최근 자산) + 우하단 플로팅 진행 패널입니다. 화면은 1개이고 상태 변형 5종(첫 방문·기본·생성중·로딩·오류)이 화면의 얼굴을 바꿉니다. 사이드바 메뉴명은 2026-07-22 개편으로 "홈"이 됐고, 스위처의 "＋ 브랜드 추가"가 모달로 열립니다.

목업 고지. 화면 속 수치(리포트 건수·점수·D-day)·리포트·썸네일은 전부 목업이며 실제 데이터가 아닙니다. 예시 브랜드는 가상 브랜드 하루온(HARUON)·글로우리프입니다. 시장 성과 수치·리뷰는 어떤 상태에서도 그리지 않습니다(증거 원칙). 유사 서비스 7종 조사 근거는 ../../research/main-dashboard-ui-references.md 입니다.
