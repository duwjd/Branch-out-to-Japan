## 0. 이 제안이 푸는 것
S2(입점 후·매출 저조)의 핵심 통증은 **리뷰(口コミ)가 안 쌓이는 것**이다(재시뮬에서 반복 확인). 그런데 현 요금표는 **리뷰 축적 운영을 상위 티어(Scale 80만)에 가둬** 두어, 30만 리포트로 "리뷰 병목"을 진단받은 S2 고객이 **바로 다음 20만 구독으로 넘어갈 이유가 플랜에 안 보인다.** → **리뷰 축적 운영을 진입 구독(Growth 20만)의 명시 항목으로 끌어내리는 것**이 이 제안의 핵심.

## 1. 현행 요금표 (as-is · 랜딩 실측)
| 티어 | 가격 | 구성 |
|---|---|---|
| FREE | 무료 | 약기법 간이 체커 등 |
| **Report** | 30만/1회 | 진단 리포트 전체(페르소나·USP) · Before&After 샘플 1건 · 약기법 진단+대체안 · 7일 내+보완 1회 |
| **Growth** ⭐ | 월 20만 | 마케팅 스튜디오(상세 전환) · 썸네일·피드 월 30콘텐츠 · 브랜드 자산 관리 · 분기 1회 리포트 재점검 |
| **Scale** | 월 80만 | Growth + 일본 기업 매칭 · 약기법 상시 모니터링 · 성과 판별 β · 전담 검수자·월 리뷰 미팅 |

**진단:** ① 리뷰 축적 운영이 어느 티어에도 명확히 없음(Scale의 "성과 판별 β"·"리뷰 미팅"은 口コミ 확보가 아님). ② Growth는 "콘텐츠 생성"만 → S2 통증(리뷰)과 연결 끊김. ③ 조기 성과 확인점 없음(P11: "리뷰 몇 개 붙는 걸 봐야 사인"). ④ 월 산출물 주기 불명(P09). ⑤ 랜딩에 가상 스탯 잔존.

## 2. 재구성 원칙
- **S2 여정:** `30만 Report(리뷰 병목 진단·설계) → 20만 Growth(리뷰 축적 실행 + 콘텐츠 + 조기 성과 확인) → 80만 Scale(상시·매칭·대량·전담)`.
- 리뷰 축적의 **설계는 Report(30만), 지속 실행·운영은 Growth(20만)**. 이 연결이 곧 구독 전환 근거.
- **실비(기프팅·인플루언서)는 전 티어 별도**(브랜드 부담). 우리는 설계·브리핑·운영지원.
- **리뷰 수·평점 보장 없음**(스테마 규제·성과보장 금지). **AI 이미지는 제안/데모, 제품 본체 컷은 브랜드 실촬영**([[../research/AI생성이미지-일본플랫폼-법규-2026-07-08]]).

## 3. 제안 티어 구성 (to-be)
| 티어 | 가격 | 구성 (🆕=신규 · ⬇=Scale에서 이동) |
|---|---|---|
| FREE | 무료 | 약기법 간이 체커 (변경 없음) |
| **Report** | 30만/1회 | 진단 리포트 전체 + **🆕 리뷰 축적 병목 진단·설계**(왜 안 쌓이나 + 초기 확보 경로 설계) + 퍼널 분해 + 약기법 실명 감사 + Before&After 실물(썸네일1·상세1) + 7일 내·보완 1회 |
| **Growth** ⭐ | 월 20만 | REPORT + **썸네일·피드 월 15~30건 + 상세 재설계 분기 1건** + **⬇🆕 리뷰 축적 운영 = "바로 실행 가능한 키트"**(기프팅 브리핑 키트·스테마 표기 가이드·완성형 리뷰 유도 문안·나노 인플루언서 **매칭 리스트+컨택 템플릿**·운영 SOP, *실비 별도*) + **🆕 첫 4주 성과 확인 리포트**(리뷰 유입·CTR·전환 전후) + 브랜드 자산 관리 + 분기 재점검 |
| **Scale** | 월 80만 | GROWTH + 약기법 **상시 모니터링**(S4) + 일본 기업 매칭(유통·커머스) + **복수 채널·대량 볼륨** + 전담 검수자·월 리뷰 미팅 + 고급 성과 분석 |

### Growth ↔ Scale 경계 (Scale이 여전히 팔리는 이유)
- **Growth = 진입 실행**: 1개 주력 상품·주채널 중심, 리뷰 축적 "시작", 표준 운영.
- **Scale = 규모·상시·위임**: 복수 채널/복수 상품 동시, 약기법 상시 모니터링(규제 민감 S4), 일본 유통 매칭, 전담 인력. → 리뷰 축적을 Growth로 내려도 Scale은 "볼륨·상시·매칭·전담"으로 차별 유지.

## 4. 확정 항목 (2026-07-09 사용자 승인)
1. ✅ **Growth 월 산출물** = **썸네일·피드 월 15~30건 + 상세 재설계 분기 1건.**
2. ✅ **리뷰 축적 운영 범위** = **"바로 실행 가능한 키트"** — 실비(샘플·인플루언서 사례비)는 별도(브랜드 부담) 그대로 두되, 우리 산출물은 "PDF 설계도"가 아니라 브랜드가 즉시 실행할 수 있는 키트(매칭 리스트·컨택 템플릿·리뷰 유도 문안·운영 SOP·첫 4주 동행)로 제공. → 도루묵(설계만 받고 실행 못 함) 방지 + 20만 마진 방어. *실비를 가격에 녹이는 것이 아님을 명확히(별개 축).*
3. ✅ **조기 성과 확인점** = "첫 4주 리뷰·CTR·전환 전후 리포트"를 Growth에 명시.
4. ✅ **실비 개략 가이드 공개** = "**마이크로 인플루언서 1인당 10만원대**" 개략치 안내에 명시.
5. ✅ **실행 경계(4차 재시뮬 후 확정)** = **20만 = self-serve 키트**(리스트·템플릿·SOP·브리핑), 실제 컨택·발송은 브랜드 or **"컨택 대행" 유료 add-on**. 실비 별도 유지.
6. ✅ **폴리시 기본값** = 콘텐츠 월 15~30건(채널·시즌) · 인플루언서 후보 리스트 월 10명 내외 · 실비 1인당 10만원대·초기 월 3~5명 권장 · 점검 첫 4주 확인 → 월 1회 라이트 체크 + 분기 심층.

## 5. 마진·리스크 고려
- 🟡 **20만에 리뷰 운영 편입 → 인건비 ↑ → 마진 압박.** 완화: (a) 실행은 **설계·브리핑 위주**(샘플 발송·컨택은 브랜드+실비별도), (b) 반자동 조립(스펙 §5.2)로 콘텐츠 한계비용 억제, (c) 리뷰 "운영지원"의 표준 범위를 SOP로 고정해 건별 확대 방지. → 20만은 여전히 **구독 획득가(트라이얼 성격)** 위상 유지(스펙 §6.1).
- 🟡 Growth가 두꺼워지면 Scale 매력 약화 우려 → §3 경계로 방어(상시·볼륨·매칭·전담).
