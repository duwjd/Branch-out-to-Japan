앱의 문과 계정의 방 — 화면 흐름

로그인은 소셜 3종(카카오·네이버·구글) + 이메일/비밀번호로 열리는 앱의 문이고, 마이페이지는 구독 플랜 가입 정보 · 계정 정보 · 결제 · 브랜드 프로필을 한곳에서 확인하는 방입니다. ⓪ 메인이 "자리만" 잡아 두었던 사이드바 계정 메뉴(MAIN-01e)가 이 두 화면으로 연결됩니다. 비회원도 앱을 둘러볼 수 있고, 리포트·썸네일 생성·브랜드 등록 직전에만 로그인 유도(게이트) 모달이 뜹니다(2026-07-23 개편 · 기획서 §3).

목업 고지. 화면 속 계정(박민정)·결제 수단·결제 내역·플랜 상태는 전부 목업이며 실제 데이터·실결제가 아닙니다. 플랜 이름·가격(FREE / Report 30만·1회 / Growth 월 20만 / Scale 월 80만)의 정본은 ../03-구독플랜-재구성-제안.md 이고, 결제 연동(PG)·구독 엔티티·결제 게이트 집행은 미구현입니다.

전체 플로우 — 문에서 방까지

비회원 접근은 더 이상 리다이렉트되지 않습니다. 앱을 둘러보다 생성·등록을 누르면 게이트 모달(기획서 §3)로 로그인이 유도됩니다(세션 만료는 여전히 로그인으로 모입니다). 로그인은 소셜·이메일 병행이고, 성공 시 기존 회원은 ⓪ 메인, 신규 회원은 온보딩(범위 외)으로 갈립니다. 마이페이지 진입은 사이드바 최하단 계정 메뉴 한 곳입니다.
