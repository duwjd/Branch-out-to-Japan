## 0. 한눈에 (TL;DR)

1. **정서적 적중은 크게 올랐는데 점수는 안 올랐다.** 20명 평균 **2.63/5** (1차 전체 평균 2.54, 1차 E안 단독 2.75). 4점 이상 0명, 3점 11명, 2점대 9명. "이거 완전 내 얘기" 반응은 1차보다 훨씬 많아졌지만(운영중 탭·색조/남성 사례가 직격), **가상 사회적 증거의 역효과와 월 구독 장벽이 상승분을 전부 상쇄**했다.
2. **최대 이탈 트리거는 "\*베타 파일럿 12개 브랜드 기준" 각주.** 20명 중 15명이 스탯 밴드·사례 배지·검수자 프로필의 별표 각주를 명시적으로 문제 삼았다. 1차의 "사회적 증거가 없다"가 이번엔 "**있는데 의심스럽다**"로 바뀌었다 — 없는 것보다 나쁠 수 있는 상태.
3. **Report 30만 원이 사실상 유일한 전환 지점.** 대조군 2명(Free만)과 P15를 제외한 17명 전원이 "Report까지는 고려"라고 답했고, 가격 저항은 사실상 사라졌다. 반면 **Growth 월 20만 원은 금액이 아니라 '결과물을 보기 전 월정액'이라는 구조 때문에 전원 보류** — "Report 받아보고 판단"이 만장일치 패턴.
4. **1차의 운영중·정체 배제 문제는 '공감' 수준에서 해소, '해결' 수준에서 미해소.** P09~P14가 전원 "운영 중·정체 탭은 내 얘기"라고 인정했지만, "그래서 리뷰(口コミ)·랭킹을 어떻게 올려주는데?"라는 질문에 페이지가 답하지 못해 점수는 2점대에 머물렀다.
5. **새로 드러난 균열: "내부 운영 툴 / 팀이 직접 쓰는" 포지션이 위임 니즈 세그먼트와 정면충돌.** 콘텐츠 리소스 부재형(P15·P16·P17, 일부 P04·P14)은 "대신 만들어달라"는 사람들인데, "직접 쓰는 스튜디오"·"생성기"라는 표현에서 "결국 내가 또 배워서 조작해야 하네"라며 이탈. P15는 히어로에서 바로 식었다.
6. **Before/After는 페이지 최강 자산으로 확정.** 20명 중 12명이 가장 끌린 섹션으로 지목("번역과 재설계의 차이를 유일하게 구체적으로 보여줌"). 카테고리가 겹칠 때 훅이 극대화됨(P03 시카 앰플 = "제 브랜드 그대로") — 반대로 자기 카테고리가 없으면(클린비건·립핸드·각질·향바디) 그 자리가 그대로 이탈 사유가 된다.

### 1-1. 점수가 말해주지 않는 것 (정성 변화)

점수 총점은 정체했지만 반응의 **질**은 달라졌다:
- 1차: "이건 내 얘기가 아니다"(배제) 가 주요 감점 사유 → 2차: "내 얘기는 맞는데 **못 믿겠다 / 결과물을 먼저 봐야겠다**"(신뢰·증거 문제)로 감점 사유가 이동.
- 즉 **메시지 적합성 문제는 대부분 풀렸고, 남은 것은 증거와 상품 구조 문제**다. 이는 다음 개선 지렛대가 카피가 아니라 ① 진짜 증거(샘플 리포트 실물, 실명 사례) ② 구독 진입 구조라는 뜻.

## 4. 공통 부정 반응 / 오해 지점 (빈도·강도순)

1. **[신뢰] "\*베타 파일럿 12개 브랜드 기준" 각주 = 페이지 전체의 신뢰 침식 (15명 언급, 최다 이탈 사유)**
   스탯 밴드(87%·+214%)→사례 배지(+38%·2.3배)→검수자 프로필→고객 인용까지 별표가 반복되며 "이 페이지의 모든 근거는 잠정적"이라는 인상을 누적시킨다. 표현은 다양하지만 반응은 동일: "12개면 표본이 너무 작다"(P03·06·08·09·13·14·18·20), "검수자가 실존 인물인지 페이지 문구인지 모르겠다"(P01·04·05·07·08·11·13·18). **1차에서 '사회적 증거 전무'가 최대 장벽이었는데, 가상 증거를 채워 넣자 '증거가 의심스러움'이라는 더 나쁜 상태가 됐다** — 특히 외주에 데인 경험이 있는 페르소나들이 "대행사 뻥카"와 즉시 연결지었다.

2. **[상품 구조] "결과물을 보기 전 월정액 불가" — Growth 구독 장벽 (11명)**
   Growth 월 20만 원 금액 자체에 대한 저항은 거의 없다(오히려 "저렴한 편" P17). 문제는 순서다: 전원이 "Report 먼저 받아보고 → 결과 보고 → 그 다음 구독 판단" 경로를 스스로 그린다. 현 요금표는 Growth를 "추천"으로 밀지만, 페르소나들은 만장일치로 Report를 관문으로 인식. **Report→Growth 전환 설계(리포트 품질이 곧 구독 영업)가 상품의 실질 퍼널.**

3. **[이해] 운영 축(일본 기업 매칭 · 성과 판별 β)은 아무도 이해 못 함 (13명)**
   "뭘 매칭해주는지, 소개인지 계약까지인지"(P13), "뭘 판별한다는 건지"(전원 공통). 이해 안 되는 블록이 3축의 1/3을 차지하며 서비스 전체의 구체성 인상을 깎는다. 입점 전 세그먼트에는 "먼 얘기"(P02는 여기서 관심 식음).

## 6. 서비스 디벨롭 인사이트 & 우선순위 액션 아이템

### 인사이트

1. **다음 지렛대는 카피가 아니라 '증거의 진위'다.** 메시지 적합성(누구의 어떤 문제인지)은 v3에서 사실상 완성됐다. 남은 감점 사유는 전부 "믿을 근거가 없다/의심스럽다"로 수렴한다. 실측 데이터가 없는 현 단계에서 취할 수 있는 정직한 증거는 두 가지뿐: **샘플 리포트 실물 공개**(전원이 "결과물을 보면 판단하겠다"고 답함 — 히어로의 "샘플 리포트 미리보기" 버튼이 실제로 그걸 보여주면 됨)와 **검수자 실명·검증 가능 프로필**.
2. **상품 퍼널은 이미 페르소나들이 그려줬다: Free(체크리스트) → Report 30만(관문) → Growth(리포트 품질 확인 후).** Growth를 "추천" 배지로 미는 현 구조보다, Report를 전환 허브로 인정하고 Report→Growth 브릿지(예: Report 비용의 Growth 첫 달 공제, 콘텐츠 3종 맛보기 포함)를 설계하는 편이 페르소나 행동 패턴과 일치한다.
3. **타겟이 실은 두 부류다: "직접 쓰고 싶은 팀" vs "맡기고 싶은 팀".** 후자(콘텐츠 리소스 부재형 4명)는 지불 여력이 있는데 "내부 툴" 포지션 문구에서 이탈한다. 내부 툴 전략을 유지하더라도, 랜딩에서 "누가 손을 움직이는지"(연결만 하면 초안·완성본이 나온다, 조작 학습 부담 없음)를 명시해야 이 세그먼트를 잃지 않는다.
4. **운영중·정체 세그먼트를 잡으려면 '리뷰 메커니즘' 서사가 필요하다.** "일본 고객이 신뢰하는 형식의 콘텐츠 → 구매 후 리뷰 작성률 → 랭킹" 같은 인과 서사와 그걸 뒷받침할 산출물(리뷰 유도 동봉 카드 문안, 리뷰 응답 문안 등)이 페이지에 한 줄이라도 있어야 공감이 전환으로 이어진다.
5. **Before/After가 영업사원이다.** 12/20이 최고 섹션으로 지목. 카테고리 적중 시 "이 사람들이 우리 카테고리를 안다"는 유일한 확신 장치로 작동. 확장(카테고리 추가) 대비 효과가 페이지 내 어떤 요소보다 확실하다.

### 우선순위 액션 아이템

**P0 — 전환을 직접 막는 것**
1. **가상 수치·증거 전면 처리:** 스탯 밴드 제거(또는 "베타 파일럿 모집 중 · 결과 공개 예정" 같은 정직한 상태 표기로 교체), 사례 배지(+38% 등) 제거, 검수자는 실명·링크 가능 프로필로 교체하거나 익명이면 "검수 기준 공개"로 대체. **현 상태는 없느니만 못하다** (15명 불신, 8명 이탈).
2. **샘플 리포트 실물 1부 공개:** "샘플 리포트 미리보기" CTA가 실제 PDF/페이지로 연결되게. 전원의 "결과물 보고 판단" 요구에 대한 유일한 정직한 답. (MVP 실행안 §12의 "샘플 진단 리포트 1개"와 동일 항목 — 최우선 제작물.)

**P1 — 세그먼트 이탈 차단**
3. **"누가 손을 움직이나" 명시:** 히어로 또는 How-it-works에 "자산 연결만 하면 초안→검수→완성본까지 도착, 배울 것 없음" 한 줄. "생성기" 표현은 "OO 초안이 도착합니다" 계열로 톤 전환 검토. (위임 니즈 4명 회수)

5. **리뷰·랭킹 해결 서사 1블록 추가:** 운영중 탭 바로 아래 "정체의 원인 → 콘텐츠 신뢰 형식 → 리뷰 전환" 인과 설명 + 관련 산출물 목록. (운영중 코어 3명)
