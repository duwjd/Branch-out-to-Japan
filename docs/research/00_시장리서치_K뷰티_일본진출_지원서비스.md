## 요약 (TL;DR)

- 시장은 **견조하나 성장세는 꺾이는 국면**. 한국이 일본 수입화장품 4년 연속 1위(점유율 30.8%)지만, 대일 수출 증가율이 **2024년 +40% → 2025년 +5.6%**로 급감. → "지원"보다 "**정착·운영 효율화**"로 니즈 이동 중.
- 지원 서비스의 **약 90%가 사람이 붙는 에이전시(운영대행·PR·인허가)**이고, **완제형 SW 툴은 거의 없음.** → 1인 개발자의 틈.
- 공통 문제는 **① 규제/약기법 통과, ② 리뷰(口コミ)/@cosme, ③ 현지화(단순 번역 ≠ 통함), ④ 멀티채널 운영 노가다** 4가지로 수렴.

## 2. 각 서비스가 푸는 핵심 문제 & 3. 공통 패턴

검증된 주장에서 도출한 4대 공통 문제:

**① 규제·약기법이 최대 병목** ✅
- 화장품은 사전 승인 불요(신고제)지만 **의약외품(미백·주름개선·자외선차단·발한억제 등)은 PMDA 사전 승인 필요, 9~12개월** 소요.
- **MAH(판매책임자) 일본 법인 필요** → 해외사는 DMAH 지정. 1인 개발자가 "대행"은 불가(라이센스 필요)하나 **정보·판별·문서화 툴**은 가능.
- 한국제 문제 사례는 성분이 배합한도·통관에서 막힘. 라벨은 일본어·전성분 내림차순 의무.

**② 리뷰(口コミ)·@cosme가 매출을 좌우** ✅
- 일본 구매결정 1위 = 리뷰. Qoo10 성공 핵심도 "口コミ 활성화". 그런데 리뷰 수집·자동·번역은 대부분 수작업.

**③ 현지화 ≠ 번역** ✅
- 한·일 마케팅 구조가 다름: 일본은 샘플링·나노/마이크로 인플루언서·상세 크리에이티브 드리븐 / 한국은 D2C·핫딜·인플루언서·검색어 KPI. "한국식 그대로"는 안 통함.

**④ 멀티채널 운영 노가다** ✅
- Qoo10/라쿠텐/아마존 각기 다른 관리화면·룰. 초기 설정부터 복잡 → 운영대행이 존재하는 이유.

> **패턴 요약**: 문제는 전부 **"반복적·규칙기반·저부가가치 운영업무인데 지금은 값비싼 사람(에이전시)이 처리"** 한다는 공통점. 규제·리뷰·현지화·채널운영 = 자동화 여지가 큰데 SW화가 거의 전무.

## 4. 시장 규모

| 지표 | 값 | 검증 |
|------|-----|------|
| 한국 화장품 **대일 수출** (2025) | **$1.087B** (+4.9%), 일본 3위 시장, 총수출의 9.5% | ✅ |
| 일본 수입시장 내 한국산 점유 | **30.8%** (141.77십억엔, 4년 연속 1위, 프랑스 22.8%) | ✅ |
| 스킨케어 / 색조 | 88.75십억엔(수입의 43.3%, +1.9% 전체) / 29.54십억엔(+8.3%) | ✅ |
| **성장 둔화** | 대일 수출 증가율 **40%(2024) → 5.6%(2025)** | ✅ |
| 한국 총 화장품 수출 | **$11.4B** (+12.2%), 세계 2위 수출국(프랑스 $24.28B 다음) | ✅ |
| 일본 화장품 시장(전체) | ~$7.9B (Mordor), 색조 서브 $3.79B→$4.75B('31) | ⚠️ 리서치사별 상이 |
| 일본 크로스보더 이커머스 | $3.01B(2022) → $5.09B(2030), CAGR 6.9% | ✅ |
| 화장품 이커머스 침투율 | 8.24% (소매평균 9.13% 미만, 드럭스토어 우세) | ⚠️ 블로그 |

**지원 서비스 SAM/SOM 추정 (자체 계산, 근거 명시)**
- 대일 진출 브랜드 매출 pool에 운영대행 수수료(5~12%)·마케팅·인허가 지출 감안 → **K-뷰티 대일 진출 지원 서비스 시장(에이전시+SaaS) SAM ≈ 연 $150M~$300M** (대일 수출 $1.09B × 운영·마케팅·규제 지출 비중 15~30%).
- 이 중 **SW/SaaS 전환 가능 SOM ≈ 초기 $5M~$15M** (현재 완제형 서비스 툴 거의 부재 → 대부분 인건비). 1인 개발자 목표는 **연 $50K~$300K ARR 달성**이 현실적.

**❌ 검증 기각(주의)**
- "Qoo10 K-뷰티 5년간 CAGR 64%·2009년 이후 10배" → 3차 만장일치 기각.
- "외국 제조소는 인정 절차를 먼저 받아야 한다" → 일반 화장품엔 부정확(의약외품 제조소만 적용), 기각.
- "약용화장품 신규 유효성분 재심사 수수료 ¥310,100 / 신청액" → 부분 확인, 낮은 신뢰도.

## 5. 1인 개발자용 미개척 서비스 Top 5 (우선순위별)

선정 기준: 규칙기반 자동화 가능 · 라이센스 불필요(정보/툴 계층) · 기존이 사람+비쌈 · SW 경쟁 희소.

### 🥇 1위 — 약기법 표시·광고 카피 자동 검수 (Compliance Copy Checker)
- **문제**: 미백/주름 등 효능 요구 문구가 약기법 위반이면 광고 리젝·판매 정지. 지금은 사람이 수동 감수.
- **왜 미개척**: Beauty Intelligence는 "성분 규제"는 하나 **광고/표시 문구(NG워드) 자동 검사는 공백**. 규칙(금지표현 리스트) 명확 → 룰+LLM 구현 쉬움.
- **구축 난이도**: ⭐⭐☆☆☆ (NG워드 DB + LLM 판정 + 대안 문구 제안. 정적 웹앱)
- **수익모델**: 브랜드/대행사 SaaS 월 $79~149 (컴플라이언스는 불황에도 지출 유지 → 검증됨).

### 🥈 2위 — 화장품 vs 의약외품 자동 분류 + 인허가 체크리스트 생성기
- **문제**: 분류 실수가 진출의 시간을 지연. MAH/서류/기간이 제품마다 다름.
- **왜 미개척**: 대행사는 유료 컨설팅으로만 제공. 사전 판별 SW 없음.
- **구축 난이도**: ⭐⭐⭐☆☆ (성분·효능 입력 → 분류 결정트리 → 맞춤 서류·기간·MAH 안내 PDF)
- **수익모델**: 무료 진단(리드) → 상세 리포트 건당 과금 + 인허가 대행사 제휴 수수료.

### 🥉 3위 — @cosme·Qoo10 리뷰(口コミ) 통합 대시보드 + 자동 알림/번역
- **문제**: 리뷰가 매출 1위 요인인데 채널별 산재·일본어 수동 부담.
- **왜 미개척**: 범용 리뷰툴 있으나 @cosme+Qoo10+한→일 특화 없음.
- **구축 난이도**: ⭐⭐⭐☆☆ (스크랩/API + 감성분석 + LLM 자동 응대·번역)
- **수익모델**: 브랜드당 월 $99~199, 리뷰량 기반 티어.

### 4위 — 일본 인플루언서 디스커버리·매칭 DB (K-뷰티 특화)
- **문제**: 브랜드가 수작업으로 인플루언서 검색(전용 DB 툴 부재 → 검증됨). 에이전시 패키지 100만~500만엔으로 고가.
- **왜 미개척**: 완제형 검색·성과추적 툴 공백, 전부 에이전시.
- **구축 난이도**: ⭐⭐⭐⭐☆ (공개 SNS 지표 수집 + 필터/매칭 UI)
- **수익모델**: 프리미엄 검색 구독 월 $149 + 캠페인 성과추적 업셀.

### 5위 — 멀티채널 상품페이지·라벨 현지화 자동 생성기
- **문제**: Qoo10/라쿠텐/아마존 각기 다른 페이지 + 일본어 라벨(전성분 내림차순·표시 표기) 반복 제작.
- **왜 미개척**: 운영대행이 수작업 대행. 성분표→규제 준수 일본어 라벨 자동 생성 SW 없음.
- **구축 난이도**: ⭐⭐⭐⭐☆ (성분 INCI 매핑 + 표기규칙 수집 + 채널별 템플릿 출력)
- **수익모델**: 제품당 생성 크레딧 + 브랜드 구독.

> **1위·2위 추천 이유**: 난이도 최저(정적/룰기반), 컴플라이언스라 지불의향·리텐션 높음, SW 경쟁 사실상 없음. **"약기법 카피 검수(1위)"를 MVP로 출발 → 인허가 체크리스트(2위)로 확장** 경로 권장.
