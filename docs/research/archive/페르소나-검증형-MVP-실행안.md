## 1. 문서 목적

이 문서는 Product Hunt와 커뮤니티 분석에서 도출된 “한국 GTM 자동화 에이전트” 가설을, MVP 팀명 **생존자들** 기준으로 재정의한 실행안이다.

초기 목표는 앱이나 SaaS를 바로 개발하는 것이 아니라, **엔비디아 / Hugging Face 기반 합성 페르소나를 활용해 일본 진출 마케팅 문제와 제안 문구의 반응 가능성을 사전 검증하는 것**이다.

이 검증은 실제 고객 인터뷰나 광고 집행을 대체하지 않는다. 대신 실제 비용을 쓰기 전, 타겟·문제·제안·가격·랜딩 메시지를 빠르게 정리하고 위험한 가정을 줄이는 **0차 검증 단계**로 사용한다.

## 2. 핵심 판단

30일 실행안은 사실상 MVP 제작에 해당한다. 다만 이 MVP는 코드 기반 제품이 아니라 **페르소나 검증형 서비스 MVP / 컨시어지 MVP**로 정의한다.

핵심 차이는 다음과 같다.

| 구분 | 일반 SaaS MVP | 생존자들 초기 MVP |
| --- | --- | --- |
| 목적 | 기능 구현 검증 | 문제·수요·메시지·지불 의사 가설 검증 |
| 1차 검증 방식 | 실제 광고, 실제 고객 유입 | 엔비디아 / Hugging Face 기반 합성 페르소나 시뮬레이션 |
| 산출물 | 앱, 대시보드, 자동화 기능 | 페르소나 반응 리포트, 랜딩 문구, 진단 패키지 초안 |
| 운영 방식 | 제품이 자동 처리 | 사람이 AI와 템플릿을 활용해 반자동 처리 |
| 장점 | 확장성 | 빠른 검증, 낮은 비용, 광고 전 리스크 축소 |
| 리스크 | 개발 범위 과대화 | 실제 시장 반응과 다를 수 있음 |

## 3. 임시 서비스 정의

### 서비스명 / 팀명

**생존자들**

### 한 줄 정의

**생존자들은 일본 진출을 준비하는 한국 뷰티 브랜드를 위해, 일본 고객 관점의 마케팅 진단·카피·콘텐츠 운영안을 7일 안에 제공하는 AI 기반 Japan Growth Studio이다.**

### 초기 포지션

- 일본어 번역 대행이 아니라, 일본 고객 관점의 메시지·콘텐츠·랜딩 구조 재설계
- AI 자동화 도구가 아니라, AI와 전문가의 반자동 운영 시스템
- 향후 SaaS화 가능한 반복 업무를 찾기 위한 검증형 서비스

## 4. 왜 광고가 아니라 페르소나 검증부터 하는가

광고 검증은 실제 시장 반응을 확인할 수 있다는 장점이 있지만, 초기에는 다음과 같은 부담이 있다.

- 광고비가 필요하다.
- 랜딩페이지와 소재를 먼저 만들어야 한다.
- 잘못된 메시지로 광고를 돌리면 비용을 낭비할 수 있다.
- 초기 타겟이 넓으면 결과 해석이 어렵다.

따라서 생존자들 MVP에서는 광고 집행 전에 **합성 페르소나 기반 사전 검증**을 먼저 수행한다.

페르소나 검증의 목적은 다음 질문에 답하는 것이다.

1. 어떤 타겟이 가장 강하게 문제를 느끼는가?
2. 어떤 문제 문구에 가장 크게 반응하는가?
3. 어떤 제안이 가장 이해하기 쉬운가?
4. 무료 진단, 샘플 리포트, 유료 패키지 중 무엇이 가장 설득력 있는가?
5. 가격 저항은 어느 구간에서 발생하는가?
6. 실제 고객 인터뷰나 광고 검증으로 넘어가기 전에 어떤 가설을 버려야 하는가?

## 5. 엔비디아 / Hugging Face 페르소나 검증 구조

### 기본 개념

엔비디아 / Hugging Face 기반의 합성 페르소나를 사용해, 실제 타겟 고객처럼 문제 상황·랜딩 문구·패키지 제안·가격에 반응하게 한다.

이때 페르소나는 단순한 “30대 여성 마케터” 수준의 인구통계형 설정으로 만들지 않는다. 실제 반응을 조금이라도 더 현실적으로 만들기 위해 다음 요소를 함께 정의한다.

- 업종
- 회사 규모
- 일본 진출 단계
- 현재 마케팅 담당자의 역할
- 예산 결정 권한
- 일본어 이해 수준
- 기존 실패 경험
- 현재 가장 급한 업무
- 외주 경험
- AI 도구 수용도
- 가격 민감도
- 리스크 우려
- 구매 트리거

### 검증 흐름

1. 타겟군별 페르소나 10~20개 생성
2. 각 페르소나에게 현재 문제 상황을 입력
3. 서비스 설명 3가지 버전을 보여줌
4. 랜딩페이지 헤드라인 5개를 비교하게 함
5. 무료 진단 / 샘플 리포트 / 유료 패키지 반응을 평가하게 함
6. 가격별 구매 가능성을 물어봄
7. 거절 이유와 신뢰 장벽을 수집
8. 반복되는 반응을 기준으로 제안 구조 수정

## 6. 검증할 초기 타겟 페르소나

### A그룹: 일본 진출 준비 코스메틱 브랜드 담당자

예시 페르소나:

- 3~15인 규모 인디 코스메틱 브랜드 운영자
- Qoo10 Japan 또는 라쿠텐 입점을 고민 중
- 일본어 번역은 가능하지만, 일본 고객에게 먹히는 표현은 잘 모름
- 상세페이지와 SNS 콘텐츠 제작에 부담을 느낌
- 외주 비용은 부담되지만, 방향성 진단에는 비용을 쓸 수 있음

검증 포인트:

- “일본어 번역만으로는 부족하다”는 메시지에 반응하는가
- 상세페이지 / SNS / 리뷰 분석 중 어떤 문제를 가장 급하게 느끼는가
- 30만 원 / 50만 원 / 100만 원 가격 중 어느 구간까지 받아들이는가

### B그룹: 일본 고객을 받고 싶은 뷰티 의료 업체 담당자

예시 페르소나:

- 피부과 / 성형외과 / 미용의료 마케팅 담당자
- 일본 고객 유입에 관심이 있지만 의료 광고 표현을 조심스러워함
- 일본어 상담 문구와 SNS 콘텐츠가 필요함
- 고가 서비스라 고객 1명 전환 가치가 큼
- 신뢰도와 전문성, 법적 리스크에 민감함

검증 포인트:

- 의료 광고 표현 리스크를 낮추는 메시지에 반응하는가
- 상담 스크립트 / 랜딩페이지 / SNS 콘텐츠 중 어떤 산출물을 더 원할까
- “AI 기반”이라는 표현이 신뢰를 높이는가, 낮추는가

### C그룹: 일본 커머스 입점 준비 소형 브랜드

예시 페르소나:

- 식품, 생활용품, 패션잡화, 뷰티 소형 브랜드 운영자
- 일본 플랫폼 입점을 알아보고 있으나, 상품 카피와 상세페이지 구조가 막힘
- 내부에 일본어 담당자가 없음
- 광고보다 먼저 상품페이지와 브랜드 메시지를 정리하고 싶어함

검증 포인트:

- “입점 전 일본향 상품 메시지 점검” 제안에 반응하는가
- 무료 체크리스트와 유료 진단 중 어떤 전환 구조가 자연스러운가
- 월 운영형보다 단발 진단형을 선호하는가

## 7. 페르소나에게 테스트할 제안 문구

### A안: 문제 자극형

> 일본 진출을 준비 중인데, 상세페이지·SNS·광고 문구를 어디서부터 바꿔야 할지 막막하신가요?
> 

### B안: 결과 제안형

> 한국 브랜드를 일본 고객이 이해할 수 있는 말과 콘텐츠로 바꿔드립니다. 일본향 마케팅 진단 리포트를 받아보세요.
> 

### C안: 자동화 강조형

> 일본어 번역만으로는 부족합니다. 상품 USP, SNS 콘텐츠, 랜딩페이지 카피까지 한 번에 정리하는 일본향 AI 마케팅 진단.
> 

### D안: 리스크 감소형

> 광고비를 쓰기 전에, 일본 고객에게 통할 메시지인지 먼저 점검하세요. 일본향 상세페이지·SNS·카피 구조를 사전 진단해드립니다.
> 

### E안: 실무 대행형

> 일본 진출 준비에 필요한 타겟 분석, 카피, SNS 콘텐츠 초안, 문의 응대 문구를 한 번에 정리해드립니다.

## 8. 페르소나 검증 질문 세트

### 문제 인식 질문

1. 현재 일본 진출 준비에서 가장 막히는 부분은 무엇인가?
2. 이 문제를 해결하지 못하면 어떤 손실이 발생하는가?
3. 내부에서 해결하기 어려운 이유는 무엇인가?
4. 기존에 외주나 번역 서비스를 써본 적이 있다면 무엇이 아쉬웠는가?

### 제안 반응 질문

1. 위 제안을 처음 봤을 때 무엇을 제공하는 서비스로 이해했는가?
2. 가장 끌리는 문구는 무엇인가?
3. 가장 애매하거나 믿기 어려운 문구는 무엇인가?
4. 이 서비스가 실제로 필요하다고 느끼는 순간은 언제인가?

### 가격 검증 질문

1. 무료 진단이라면 신청할 의향이 있는가?
2. 30만 원 진단 리포트라면 구매할 의향이 있는가?
3. 50만 원 Starter Kit이라면 구매할 의향이 있는가?
4. 100만 원 상세 진단 + 콘텐츠 초안 패키지라면 구매할 의향이 있는가?
5. 월 100만 원~250만 원 운영 패키지라면 어떤 조건에서 검토할 것인가?

### 신뢰 장벽 질문

1. 구매를 망설이게 하는 가장 큰 이유는 무엇인가?
2. 어떤 사례나 샘플이 있으면 신뢰가 올라가는가?
3. “AI 기반”이라는 표현은 신뢰에 도움이 되는가, 방해가 되는가?
4. 사람 전문가의 개입이 어느 정도 필요하다고 느끼는가?

## 10. 검증 지표

페르소나 검증에서는 실제 클릭률이나 전환율 대신 다음 지표를 본다.

| 확인 항목 | 의미 |
| --- | --- |
| 문제 공감도 | 타겟이 해당 문제를 실제로 중요하게 여기는가 |
| 문구 이해도 | 제안이 한 번에 이해되는가 |
| 긴급도 | 지금 해결해야 하는 문제로 느끼는가 |
| 산출물 선호도 | 어떤 결과물을 가장 원한다고 말하는가 |
| 가격 수용도 | 얼마까지 지불 의향이 있는가 |
| 신뢰 장벽 | 구매를 막는 핵심 불안 요소는 무엇인가 |
| 다음 행동 의향 | 무료 진단, 상담, 유료 진단 중 어디까지 움직이는가 |

### 해석 기준

- 문제 공감도는 높은데 가격 수용도가 낮으면 무료 콘텐츠 또는 저가 진단 상품부터 시작한다.
- 가격 수용도는 있는데 신뢰 장벽이 높으면 샘플 리포트와 사례를 먼저 만든다.
- AI 표현에 거부감이 있으면 “AI 기반”보다 “일본향 전문가 진단 + 자동화 리포트”로 표현한다.
- 코스메틱, 의료, 커머스 중 특정 타겟만 강하게 반응하면 초기 타겟을 좁힌다.

6. 피드 / 스토리 / 블로그 콘텐츠 템플릿
7. 상담 / 문의 응대 일본어 문안
8. 주간 실행 체크리스트
9. AI 기반 반자동 운영 워크플로 일부 제공

#### 가격 검증안

- 무료: 일본 진출 마케팅 체크리스트
- 입문형: 30만 원 진단 리포트
- 기본형: 50만 원 Starter Kit
- 확장형: 100만 원 상세 진단 + 콘텐츠 초안 패키지
- 운영형: 월 100만 원~250만 원 리테이너

## 13. 페르소나 검증의 한계

합성 페르소나는 실제 고객이 아니다. 따라서 다음 한계를 반드시 전제로 한다.

1. 실제 구매 행동을 완전히 예측할 수 없다.
2. 모델이 그럴듯한 답을 만들 수 있다.
3. 타겟 데이터가 부정확하면 검증 결과도 왜곡된다.
4. 인구통계 정보만으로 만든 페르소나는 반응 예측력이 낮을 수 있다.
5. 최종 검증은 반드시 실제 고객 인터뷰, 신청 폼, 유료 제안으로 이어져야 한다.

따라서 생존자들 MVP에서 페르소나 검증은 **광고 전 사전 검토**로만 사용한다. 최종 사업성 판단은 실제 고객 반응으로 확인한다.

## 14. 다음 작업 순서

1. 코스메틱 / 의료 / 커머스 중 초기 타겟 우선순위를 정한다.
2. 타겟별 페르소나 생성 기준표를 만든다.
3. 엔비디아 / Hugging Face 기반 합성 페르소나를 10~20개 생성한다.
4. 제안 문구 A~E안을 페르소나에게 테스트한다.
5. 가격별 반응을 기록한다.
6. 가장 반응이 좋은 타겟과 제안 문구를 선택한다.
7. 샘플 진단 리포트와 랜딩페이지 초안을 만든다.
8. 실제 고객 5~10명에게 인터뷰 또는 무료 진단을 제안한다.

## 15. 최종 요약

생존자들의 첫 MVP는 광고 집행형 MVP가 아니라 **엔비디아 / Hugging Face 기반 합성 페르소나 검증형 MVP**다.

초기에는 넓은 한국 GTM 자동화 시장을 노리지 않는다. 대신 일본 진출을 준비하는 한국 뷰티 브랜드의 구체적인 문제, 즉 **일본 고객에게 통하는 메시지·콘텐츠·랜딩페이지·SNS 운영안 부족**을 해결하는 좁은 패키지로 시작한다.

이 MVP의 목적은 완성된 자동화 도구를 판매하는 것이 아니라, 다음 세 가지를 검증하는 것이다.

1. 어떤 타겟이 일본 진출 마케팅 문제를 가장 강하게 느끼는가?
2. 어떤 제안 문구와 산출물이 가장 설득력 있는가?
3. 실제 고객 인터뷰와 유료 제안으로 넘어갈 만한 가설이 무엇인가?

이 세 가지가 확인되면, 생존자들은 이후 실제 고객 인터뷰 → 무료 진단 → 유료 패키지 → 서비스형 운영 → 반복 기능 기반 SaaS로 확장할 수 있다.
