---

로드맵 대비 진척을 요약한다.

1. `docs/04-roadmap.md` 의 Phase별 체크리스트를 읽는다.
2. 실제 코드/문서 상태(`app/`, `docs/`, `design/`, git 최근 변경)와 대조한다.
3. 다음을 표로 정리한다:
   - Phase별 완료/진행/미착수 항목
   - 막혀 있는(blocked) 항목과 원인(열린 결정 등)
   - 지금 착수하기 좋은 다음 작업 1~3개와 담당 에이전트 제안

체크리스트 상태가 실제와 다르면 `docs/04-roadmap.md` 갱신을 제안한다.
